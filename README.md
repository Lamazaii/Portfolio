Portfolio – Vianney Biston

Bienvenue sur le dépôt de mon Portfolio personnel, développé en HTML /
CSS / JavaScript. Ce site a pour objectif de présenter mes projets, mes
compétences et mon parcours en informatique.


Lien du portfolio en ligne : https://lamazaii.github.io/Portfolio/


Fonctionnalités - Présentation de mon profil  - Liste
de mes projets avec descriptions et liens GitHub - Design responsive
(mobile, tablette, desktop) - Interface développée entièrement en HTML,
CSS et JavaScript - Navigation intuitive et animations légères


Technologies utilisées - HTML5 - CSS3 - JavaScript - Git &
GitHub


Auteur : Vianney Biston
